# vue-dependency-demo

[![Greenkeeper badge](https://badges.greenkeeper.io/adamchenwei/vue-dependency-demo.svg)](https://greenkeeper.io/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
